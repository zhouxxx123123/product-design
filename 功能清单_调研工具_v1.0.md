# 中科琉光调研工具 · 功能清单 v1.0

更新日期：2026-03-09 · 共 81 个功能点 · 前端呈现 57 个 · 纯后端 24 个
角色：ADMIN（管理员）/ SALES（销售）/ EXPERT（专家）· 所有功能 Web + Android App 同步实现

**产品理念 — OpenClaw**：每个页面右下角内嵌浮动 Copilot 对话框，点击唤起后可用自然语言执行任意系统操作。传统页面路径完整保留，两种方式并行。
